# Experimente mit Arduino UNO R4

Experimente mit dem Board **Arduino UNO R4**.

Die Experimente sollen Erfahrung mit der Programmierung von ARM Mikrocontroller bringen.

Hardwarenahe Programmierung in Assembler ohne Bibliotheken soll helfen die Details der
Mikrocontroller Programmierung zu verstehen.
Die Kombination aus einer höheren Programmiersprachen (C, Go oder LUA) mit Assembler soll folgen.
